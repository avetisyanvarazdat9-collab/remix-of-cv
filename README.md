## 🇬🇧 English
This is a personal portfolio/website built with modern serverless architecture.
- **Technologies:** Lovable, GitHub, Vercel, Supabase.
- **Workflow:** Every change in Lovable is automatically synced to GitHub and deployed via Vercel.

## ⚙️ Project Management
- **Frontend:** Lovable (React/Tailwind)
- **Database:** Supabase (PostgreSQL)
- **Deployment:** Vercel

## 🛠 Workflow (Աշխատանքային հերթականություն)
1. **Develop:** Make changes in Lovable.
2. **Sync:** Click "Sync with GitHub".
3. **Deploy:** Vercel automatically deploys the latest version.
4. **Database:** Run SQL scripts in Supabase SQL Editor if schema changes.

## 🆘 Troubleshooting
If the site fails to load:
1. Check Vercel logs.
2. Run pending SQL migrations in Supabase.
3. Perform a manual Redeploy in Vercel if necessary.
